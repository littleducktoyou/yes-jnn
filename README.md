# yes jnn

yes jnn is an open-source markdown notes web app modeled after Joplin.

> This is a vibe shitted app. I have spent more time writing markdowns than I have promting llms. That should tell you everything about this project.

## Stack

- **[TanStack Start](https://tanstack.com/start)** — full-stack SSR React framework with file-based routing
- **[Supabase](https://supabase.com)** — email/password auth + PostgreSQL with Row Level Security on every table
- **[Tailwind CSS v4](https://tailwindcss.com)** — utility-first styling, no custom CSS build step
- **[shadcn/ui](https://ui.shadcn.com)** — accessible Radix UI primitives, pre-wired to Tailwind
- **[Bun](https://bun.sh)** — package manager & runtime; replaces Node + npm for install & dev server

## Features

- Notebooks for grouping notes
- Markdown priview
- Email & password authentication. I recommend using a [duck email](https://duckduckgo.com/email) (I don't know what I am doing)
